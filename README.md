# fastapi-series

## 功能说明
- fastapi基础知识
- 多app目录设计

### fastapi基础知识
- 路径参数与数值校验
- 查询参数与字符串校验
- Request Body
- Request Cookie/Header/Form/File
- 响应模型
- 响应状态码
- 依赖注入系统
- 中间价
- ...

### 多app目录设计
- 适用于较复杂的业务
- 模块进行解耦

## 运行
#### 1、安装依赖
```shell
pip install -r requirements.txt
```
#### 2、运行
```python
python main.py
```
